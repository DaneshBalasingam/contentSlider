Welcome to Content Slider PlugIn

This is for educational purposes only.  

Easy to use. See demo index.html

Each carousel item needs eg
<div data-item="0" class="panel current">the item</div>

and a button for item as below

<span data-slide-to="0" class="dot"> &bull;</span>

Default option for auto scrolling is true. To turn off pass arg

autoScrolling: false

See demo index.html
